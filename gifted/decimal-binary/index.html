10진법과 2진법 변환 실험실

10진수와 2진수를 서로 바꾸는 과정을 직접 해 보며, 자리값과 2의 거듭제곱 규칙을 탐구하는 웹앱입니다.

활동 순서

1
2진수의 각 자리가 어떤 값을 뜻하는지 살펴봅니다.
2
10진수를 2진수로 바꾸는 과정을 직접 따라 해 봅니다.
3
2진수를 10진수로 다시 바꾸며 규칙을 확인합니다.
4
퀴즈를 풀며 자리값 규칙을 스스로 정리합니다.

1. 자리값 탐구

아래 비트를 눌러 0과 1을 바꾸면, 각 자리의 값이 더해져 10진수가 됩니다.

현재 2진수: 00000000
현재 10진수: 0

2. 10진수 → 2진수

10진수를 입력하고, 나누기 2 과정을 보며 2진수로 바꾸어 보세요.

숫자를 입력하고 변환하기를 눌러 보세요.
결과 2진수: -

3. 2진수 → 10진수

2진수를 입력하면 각 자리의 값이 어떻게 더해지는지 볼 수 있습니다.

2진수를 입력하고 변환하기를 눌러 보세요.
결과 10진수: -

4. 변환 퀴즈

10진수 → 2진수
문제: 13
문제를 보고 답을 입력해 보세요.
