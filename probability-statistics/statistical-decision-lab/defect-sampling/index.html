검사한 표본 아직 직접 보지 않은 제품 980개는 검사 결과만으로 직접 확인되지 않았습니다.

미발견 확률 계산

Q1 계산 확인
( 1 - p ) n

각 제품이 정상일 확률을 표본 수만큼 곱해, 표본에서 불량이 0개 나오는 경우를 봅니다.

곱해지는 구조

p와 n을 입력해 계산하면, n개의 독립 시행이 어떻게 누적되는지 함께 확인합니다.

내 미발견 확률 확인

p와 n을 입력하면 입력값에 대한 미발견 확률만 계산합니다.

미발견
