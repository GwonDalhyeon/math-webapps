# 작업지시서 보관 규칙

이 폴더는 Codex가 작성한 공개 가능한 작업지시서와 교육용 기획서를 보관하는 곳이다.

Claude Code에 구현을 맡길 때는 작업지시서를 이 폴더에 저장하고, Claude Code가 해당 파일을 먼저 읽도록 지시한다.

## 파일 이름

- `YYYY-MM-DD-topic-name.md` 형식을 권장한다.
- 폴더명과 파일명은 영어 소문자, 숫자, 하이픈을 사용한다.

예:

```text
2026-05-20-sequence-limit.md
2026-05-20-binomial-distribution.md
```

## 포함할 내용

```text
웹앱 이름:
위치:
대상 학생:
학습 목표:
핵심 개념:
사용자 조작:
시각화 요소:
상태/결과 표시:
수학 모델 또는 계산식:
생성할 파일:
수정할 파일:
수정하지 말 것:
완료 기준:
검증 방법:
```

## 주의

- 공개 저장소에 올라가도 되는 내용만 둔다.
- API 키, 계정 정보, 학생 개인정보, 비공개 URL은 쓰지 않는다.
- 공개하면 안 되는 임시 작업지시서는 `.work/claude-tasks/` 또는 `tasks/private/`에 둔다.
