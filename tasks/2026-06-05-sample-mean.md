# 표본평균의 분포 실험실 웹앱 작업지시서

웹앱 이름:
표본평균의 분포 실험실

위치:
`probability-statistics/sample-mean/index.html`

참고자료:
`probability-statistics/표본평균.html`

대상 학생:
고등학교 확률과 통계 과목에서 모집단, 표본, 표본평균, 모평균 추정, 중심극한정리를 배우는 일반계 고등학교 학생

수업 상황:
교사는 "한 번 뽑은 표본의 평균은 매번 흔들리는데, 여러 번 뽑은 표본평균들은 어떤 분포를 만들까?"라는 질문으로 수업을 시작한다. 학생은 모집단 모양과 표본 크기를 바꾸고, 표본을 반복 추출하면서 표본평균의 분포가 모평균 주변에 모이고 표준오차가 `sigma / sqrt(n)`에 가까워지는 과정을 관찰한다.

학습 목표:
- 모집단과 표본, 표본평균의 차이를 시각적으로 구분한다.
- 표본을 한 번 뽑을 때마다 표본평균이 달라진다는 점을 확인한다.
- 한 표본에서 계산한 표본평균은 `값 하나`이고, 이 값을 여러 번 모으면 새로운 분포, 즉 표본평균의 분포가 만들어짐을 이해한다.
- 표본평균들의 평균이 모평균에 가까워지는 경향을 관찰한다.
- 표본 크기 `n`이 커질수록 표본평균의 분포가 좁아지고, 그 표준편차가 이론값 `sigma / sqrt(n)`에 가까워짐을 확인한다.
- 모집단이 정규분포가 아니어도 표본 크기가 충분히 크면 표본평균의 분포가 종 모양에 가까워지는 중심극한정리의 의미를 직관적으로 이해한다.
- 표본 크기 `n`과 표본을 반복 추출한 횟수를 혼동하지 않는다.

핵심 개념:
- 모집단
- 표본
- 표본평균
- 모평균
- 모표준편차
- 표본평균의 분포
- 표준오차
- 중심극한정리

## 개념 이해 우선순위

이 웹앱은 중심극한정리를 바로 설명하기 전에 다음 세 단계를 먼저 분명히 보여주어야 한다.

1. 모집단은 전체 값들의 분포이다.
2. 표본은 모집단에서 뽑은 `n`개의 값이며, 표본평균은 그 표본에서 계산된 `값 하나`이다.
3. 표본을 여러 번 반복해서 뽑으면 표본평균도 여러 개 생기고, 이 표본평균들이 모여 `표본평균의 분포`를 만든다.

학생이 특히 헷갈리기 쉬운 구분:

- `표본의 분포`: 한 번 뽑은 표본 안의 원자료 `n`개가 어떻게 퍼져 있는지 보여준다.
- `표본평균의 분포`: 표본을 여러 번 뽑아 얻은 여러 개의 `xbar`가 어떻게 퍼져 있는지 보여준다.
- `표본 크기 n`: 한 표본 안에 들어가는 값의 개수이다.
- `반복 추출 횟수`: 표본을 몇 번 만들었는지, 즉 표본평균이 몇 개 쌓였는지를 뜻한다.

따라서 화면에는 `현재 표본 크기 n`과 `누적 표본 수`를 서로 다른 위치와 다른 문구로 표시한다.

## 설계 원칙

이 웹앱은 결과 숫자를 계산하는 도구가 아니라 다음 흐름을 보여주는 실험실이어야 한다.

```text
모집단 분포 확인 -> 표본 n개 추출 -> 표본평균 xbar 계산 -> xbar 1개 누적 -> 표본평균의 분포 해석
```

화면의 핵심은 세 개의 히스토그램이다.

1. 모집단 분포
2. 최근 추출된 표본
3. 표본평균의 분포

학생은 "모집단 하나에서 표본을 여러 번 뽑으면 표본평균들이 따로 분포를 만든다"는 점을 한 화면에서 읽을 수 있어야 한다.

`1회 추출`을 눌렀을 때는 다음 변화가 순서대로 보이거나, 최소한 즉시 확인되어야 한다.

1. 최근 표본 영역에 `n`개의 값이 표시된다.
2. 그 표본의 평균 `xbar`가 계산되어 선 또는 점으로 표시된다.
3. 표본평균의 분포 영역에 방금 계산한 `xbar` 하나가 새로 추가된다.

이 연결이 보이지 않으면 학생은 세 번째 그래프가 무엇을 모은 것인지 이해하기 어렵다.

## 사용자 조작

### 모집단 분포 선택

`select` 요소로 제공한다.

- 정규분포
  - 평균 100, 표준편차 15
  - Box-Muller 변환으로 생성
- 균등분포
  - 50부터 150 사이 균등분포
- 오른쪽으로 치우친 분포
  - 지수분포를 변환해 평균이 대략 100 근처가 되도록 생성
- 두 봉우리 분포
  - 평균 80 근처와 120 근처의 두 정규분포를 섞어서 생성
  - 중심극한정리가 더 잘 드러나는 비교용 분포

### 표본 크기 `n`

- 슬라이더와 숫자 표시를 함께 둔다.
- 범위: `2`부터 `100`
- 기본값: `10`
- 빠른 선택 버튼:
  - `n = 2`
  - `n = 5`
  - `n = 10`
  - `n = 30`
  - `n = 50`

표본 크기를 바꾸면 기존 표본평균 누적값은 초기화한다. 서로 다른 `n`의 표본평균을 한 분포에 섞지 않는다.

표본 크기 설명 문구를 조작 패널 안에 짧게 넣는다.

```text
n은 한 번에 뽑는 값의 개수입니다. n을 바꾸면 표본평균의 흔들림이 달라집니다.
```

### 반복 추출 횟수

반복 추출 횟수는 `n`과 다르다는 점을 화면에서 명시한다.

- `1회 추출`: 표본 1개를 만들고, 표본평균 1개를 누적한다.
- `10회 추출`: 표본 10개를 만들고, 표본평균 10개를 누적한다.
- `100회 추출`: 표본 100개를 만들고, 표본평균 100개를 누적한다.
- `1000회 추출`: 표본 1000개를 만들고, 표본평균 1000개를 누적한다.

반복 횟수 설명 문구:

```text
반복 추출 횟수는 표본평균을 몇 개 쌓을지 정합니다.
```

### 실행 버튼

반복 실험 앱이므로 다음 버튼을 둔다.

- `1회 추출`
- `10회 추출`
- `100회 추출`
- `1000회 추출`
- `자동 실행`
- `일시정지`
- `초기화`

자동 실행은 중복 타이머가 생기지 않게 한다. 이미 자동 실행 중이면 `자동 실행` 버튼을 다시 눌러도 새 타이머를 추가하지 않는다.

### 핵심 수치 요약

카드 또는 가로 요약줄 형태로 표시한다.

- 추출한 표본 수
- 표본 크기 `n`
- 모평균 `mu`
- 모표준편차 `sigma`
- 최근 표본평균 `xbar`
- 표본평균들의 평균
- 표본평균들의 표준편차
- 이론적 표준오차 `sigma / sqrt(n)`

### 세 개의 시각화 영역

각 영역에는 제목, 캔버스, 간단한 수치 설명을 둔다.

1. 모집단 분포
   - 모집단 전체 데이터의 히스토그램
   - 모평균 위치에 세로선 표시

2. 최근 추출된 표본
   - 마지막으로 뽑은 표본 데이터의 히스토그램 또는 점 표시
   - 최근 표본평균 위치에 세로선 표시
   - 표본 크기가 작을 때는 막대보다 점 또는 러그 표시가 더 잘 보이면 함께 사용해도 된다.
   - 제목 또는 보조 문구에 "이 그래프는 표본 안의 원자료 n개"라고 명시한다.

3. 표본평균의 분포
   - 누적된 표본평균들의 히스토그램
   - 모평균 위치에 기준선 표시
   - 표본평균들의 평균 위치에 세로선 표시
   - 가능하면 `mu ± sigma / sqrt(n)` 위치에 점선 표시
   - 방금 추가된 최신 `xbar`를 점, 세로선, 또는 잠깐 강조되는 막대로 표시한다.
   - 제목 또는 보조 문구에 "이 그래프는 여러 번 얻은 xbar 값들"이라고 명시한다.

### 1회 추출 흐름 표시

세 그래프 아래 또는 핵심 수치 요약 근처에 짧은 흐름 표시를 둔다.

```text
이번 표본: n개 값 -> 이번 표본평균 xbar = 00.00 -> 표본평균 분포에 1개 추가
```

이 영역은 `1회 추출`을 누를 때마다 갱신한다. `10회`, `100회`, `1000회` 추출을 누른 경우에는 마지막 표본을 기준으로 표시하되, 누적된 표본평균 개수가 함께 바뀌어야 한다.

### 현재 해석

외부 AI 호출 없이, 현재 실험 상태를 바탕으로 문장을 자동 생성한다.

예시 문구:

- 처음 상태:
  - "왼쪽은 모집단 전체의 분포입니다. 가운데는 한 번 뽑은 표본 안의 값들이고, 오른쪽은 여러 번 얻은 표본평균 xbar들이 쌓일 자리입니다."
- 1회 이상 추출했을 때:
  - "이번 표본에서 원자료 n개를 평균 내어 xbar 하나를 만들었습니다. 그 xbar 하나가 오른쪽 분포에 새로 들어갑니다."
- 표본 수가 적을 때:
  - "아직 표본평균이 많이 쌓이지 않았습니다. 100회 이상 추출하면 분포 모양을 더 안정적으로 볼 수 있습니다."
- 표본평균의 평균이 모평균에 가까울 때:
  - "표본평균들의 평균이 모평균에 가까워지고 있습니다. 여러 표본을 반복해서 뽑으면 중심이 모평균 주변에 놓입니다."
- 표준오차가 이론값과 가까울 때:
  - "표본평균의 표준편차가 이론값 sigma / sqrt(n)에 가까워지고 있습니다."
- 표본 크기가 클 때:
  - "n이 커져서 한 표본 안에 더 많은 값이 들어가므로, 표본평균의 흔들림이 줄어듭니다."
- 치우친 분포나 두 봉우리 분포를 선택했을 때:
  - "모집단 모양은 정규분포가 아니지만, 표본평균의 분포는 반복 추출이 쌓일수록 더 매끄러운 종 모양에 가까워집니다."

## 시각화 요소

- Canvas로 직접 그린다.
- 외부 차트 라이브러리를 사용하지 않는다.

- 모집단: 파랑 계열
- 최근 표본: 보라 또는 자주 계열
- 표본평균의 분포: 초록 계열
- 모평균 기준선: 빨간 실선
- 최근 표본평균 또는 표본평균들의 평균: 진한 선
- 이론적 표준오차 범위: 회색 또는 주황 점선

색상은 프로젝트 전체가 지나치게 한 색 계열로 보이지 않도록 균형 있게 쓴다.

### 표본평균 분포 보조 표시

표본평균의 분포 캔버스에는 다음을 표시한다.

- 모평균 `mu` 기준선
- 표본평균들의 평균 기준선
- `mu - sigma / sqrt(n)`, `mu + sigma / sqrt(n)` 점선
- 최신 표본평균 `latestMean` 강조 표시
- 표본평균 개수가 충분히 많을 때, 선택적으로 정규곡선 근사선을 얇게 표시할 수 있다.

정규곡선 근사선은 외부 라이브러리 없이 직접 계산한다.

### 오개념 방지 표시

학생이 표본의 분포와 표본평균의 분포를 혼동하지 않도록 각 그래프 아래에 짧은 라벨을 둔다.

- 모집단 분포:
  - "전체 값들의 분포"
- 최근 추출된 표본:
  - "이번에 뽑은 원자료 n개의 분포"
- 표본평균의 분포:
  - "반복 추출로 얻은 xbar 값들의 분포"

표본평균의 분포 그래프에는 원자료를 직접 넣지 않는다. 반드시 각 표본에서 계산한 `xbar`만 누적한다.

## 상태/결과 표시

결과 패널에는 다음 값을 표시한다.

- 모집단 크기: `10000`
- 표본 크기: 현재 `n`
- 누적 표본 수: `sampleMeans.length`
- 모평균: `mu`
- 모표준편차: `sigma`
- 최근 표본평균: `latestMean`
- 최근 표본 표준편차: `latestSampleStd`
- 표본평균들의 평균: `mean(sampleMeans)`
- 표본평균들의 표준편차: `std(sampleMeans)`
- 이론적 표준오차: `sigma / sqrt(n)`
- 실험값과 이론값의 차이:
  - `abs(std(sampleMeans) - sigma / sqrt(n))`
- 최근 흐름 문장:
  - `n개 값 -> xbar 1개 -> 누적 표본평균 수 +1`

표시는 소수 둘째 자리까지를 기본으로 한다.

## 수학 모델 또는 계산식

모집단:

- 브라우저에서 크기 `10000`의 유한 모집단을 생성한다.
- 모집단 분포를 바꾸면 모집단을 새로 생성하고, 표본평균 누적값도 초기화한다.

표본 추출:

- 복원추출로 구현한다.
- 한 표본은 모집단에서 `n`개의 값을 무작위로 뽑은 배열이다.
- `runSampling(1)`은 표본 배열 하나를 만들고 표본평균 하나를 `sampleMeans`에 추가한다.
- `runSampling(100)`은 이 과정을 100번 반복하므로 표본평균 100개가 추가된다.

표본평균:

```text
xbar = (x1 + x2 + ... + xn) / n
```

모평균:

```text
mu = population의 평균
```

모표준편차:

```text
sigma = sqrt(sum((xi - mu)^2) / N)
```

표본 표준편차:

```text
s = sqrt(sum((xi - xbar)^2) / (n - 1))
```

표본평균의 분포:

```text
sampleMeans = [xbar_1, xbar_2, ..., xbar_m]
```

여기서 `m`은 반복 추출 횟수이다. `m`은 표본 크기 `n`과 다르다.

표본평균의 표준편차 실험값:

```text
std(sampleMeans)
```

이론적 표준오차:

```text
sigma / sqrt(n)
```

중심극한정리 연결:

```text
n이 충분히 크면 xbar의 분포는 대략 정규분포에 가까워진다.
평균은 mu, 표준편차는 sigma / sqrt(n)에 가까워진다.
```

## 교사용 시연 흐름

1. 기본값 `정규분포`, `n = 10`으로 시작한다.
2. `1회 추출`을 여러 번 눌러 최근 표본과 표본평균이 매번 달라지는 것을 보여준다.
3. 각 `1회 추출`마다 오른쪽 그래프에 `xbar`가 하나씩 쌓인다는 점을 확인한다.

5. 표본평균들의 평균이 모평균에 가까운지 확인한다.
6. 표본평균의 표준편차와 이론값 `σ/√n`을 비교한다.
7. `n = 2`, `n = 10`, `n = 30`, `n = 50`으로 바꾸며 표본평균 분포의 폭이 줄어드는 것을 비교한다.
8. 모집단을 `오른쪽으로 치우친 분포`나 `두 봉우리 분포`로 바꾸고, 모집단 모양과 표본평균 분포 모양의 차이를 관찰한다.
9. "모집단 모양과 표본평균 분포 모양은 같지 않을 수 있다"는 점과 중심극한정리를 연결한다.

## 검증 방법

기본 동작:

- 페이지를 열었을 때 모집단 분포가 바로 보이는지 확인한다.
- 기본값이 `정규분포`, `n = 10`인지 확인한다.
- `1회 추출`을 누르면 최근 표본과 최근 표본평균이 갱신되는지 확인한다.
- `1회 추출`을 한 번 누르면 누적 표본 수가 정확히 1 증가하는지 확인한다.
- `1회 추출`을 한 번 누를 때 표본평균의 분포에는 원자료 `n`개가 아니라 `xbar` 1개만 추가되는지 확인한다.
- `1000회 추출` 후 표본평균의 분포가 충분히 쌓이는지 확인한다.

수치 검증:

- 정규분포 기본 모집단에서 모평균이 대략 `100`, 모표준편차가 대략 `15` 근처인지 확인한다.
- `n = 10`일 때 이론적 표준오차가 대략 `15 / sqrt(10) ≈ 4.74` 근처인지 확인한다.
- `1000회` 이상 추출 후 표본평균들의 표준편차가 이론값 근처인지 확인한다.
- `n = 50`으로 바꾸면 이론적 표준오차가 `n = 10`일 때보다 작아지는지 확인한다.
- 같은 모집단에서 `n = 2`와 `n = 50`을 각각 `1000회` 이상 추출했을 때, `n = 50`의 표본평균 분포가 더 좁은지 확인한다.

상태 검증:

- 자동 실행 중 `초기화`를 누르면 자동 실행이 멈추는지 확인한다.
- 자동 실행 버튼을 여러 번 눌러도 속도가 비정상적으로 빨라지지 않는지 확인한다.
- 모집단 분포를 바꾼 뒤 이전 표본평균 분포가 남지 않는지 확인한다.
- 표본 크기 `n`을 바꾼 뒤 이전 표본평균 분포가 남지 않는지 확인한다.

레이아웃 검증:

- 데스크톱 폭에서 세 시각화 영역이 한 줄 또는 보기 좋은 그리드로 배치되는지 확인한다.
- 모바일 폭에서 세 시각화 영역이 한 열로 쌓이고 텍스트가 겹치지 않는지 확인한다.
- 버튼 텍스트가 부모 버튼 밖으로 넘치지 않는지 확인한다.

콘솔 검증:

- 개발자 도구 콘솔에 `Cannot read properties of null`, `undefined`, 네트워크 API 오류가 없어야 한다.
- 외부 API 호출이 없어야 한다.
