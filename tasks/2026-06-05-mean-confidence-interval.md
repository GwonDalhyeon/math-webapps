# 표본으로 모평균 추정하기 웹앱 작업지시서

웹앱 이름:
표본으로 모평균 추정하기

위치:
`probability-statistics/mean-confidence-interval/index.html`

관련 웹앱:
- `probability-statistics/sample-mean-box/index.html`
- `probability-statistics/sample-mean/index.html`

이 작업지시서는 모평균의 추정을 다루는 별도 웹앱을 만들기 위한 것이다. 앞의 두 웹앱이 표본평균이 만들어지는 과정과 표본평균의 분포를 보여준다면, 이 웹앱은 `모집단 정보를 모르는 상황에서 표본 하나로 모평균 μ의 위치를 추정하는 과정`을 보여준다.

검토 후 수정 원칙:
이 웹앱은 `한 번 뽑은 표본으로 구간을 만드는 화면`과 `같은 방법을 반복했을 때 포함 비율을 보는 화면`을 구분해야 한다. 단일 신뢰구간 화면에서는 "이번 구간에 μ가 몇 % 확률로 들어 있다"는 인상을 주지 않는다. 반복 실험 화면에서만 신뢰수준을 장기적 포함 비율과 연결한다.

대상 학생:
고등학교 확률과 통계 과목에서 모평균의 추정, 표본평균, 표본표준편차, 신뢰구간, 신뢰도를 배우는 일반계 고등학교 학생

수업 상황:
교사는 "모집단 전체는 조사할 수 없고, 표본만 볼 수 있다면 모평균은 어디쯤 있다고 말할 수 있을까?"라는 질문으로 수업을 시작한다. 학생은 표본에서 계산한 표본평균 `xbar`와 표본표준편차 `s`만 사용해 모평균의 신뢰구간을 만든다. 이후 같은 방법을 여러 번 반복해 만든 구간들을 보며 신뢰도 또는 신뢰수준이 장기적 포함 비율과 어떤 관계가 있는지 관찰한다.

학습 목표:
- 모평균 추정은 모집단 전체 정보를 모르는 상황에서 시작된다는 점을 이해한다.
- 추정할 때 사용할 수 있는 정보는 표본 크기 `n`, 표본평균 `xbar`, 표본표준편차 `s`뿐임을 확인한다.
- 표본평균 `xbar`는 모평균 `μ`의 점추정값임을 이해한다.
- 표본표준편차 `s`와 표본 크기 `n`으로 표준오차 추정값 `s / sqrt(n)`을 계산한다.
- 신뢰수준이 높아질수록 오차한계가 커져 신뢰구간이 넓어진다는 것을 확인한다.
- 신뢰수준은 특정 구간 하나에 대한 확률이라기보다, 같은 방법으로 반복해 만든 구간들이 실제 `μ`를 포함하는 장기적 비율이라는 점을 이해한다.
- 실제 모평균 `μ`는 추정 과정에서는 보이지 않고, 구간을 만든 뒤 사후 확인용으로만 공개된다는 점을 이해한다.

핵심 개념:
- 모집단 정보의 미지성
- 표본
- 표본평균
- 표본표준편차
- 표준오차 추정값
- 오차한계
- 신뢰구간
- 신뢰수준
- 장기적 포함 비율

## 화면 모드 구분

이 앱은 두 가지 모드를 명확히 구분한다.

### 1. 단일 표본 추정 모드

목표:
표본 하나로 신뢰구간 하나를 만드는 과정을 보여준다.

이 모드에서 보여줄 것:

- 표본값
- `xbar`
- `s`
- `s / sqrt(n)`
- 신뢰도에 따른 오차한계
- 이번 표본으로 만든 신뢰구간 `[L, U]`

이 모드에서 숨길 것:

- 실제 `μ`
- 실제 `σ`
- 모집단 전체 분포

중요:
단일 표본 추정 모드에서는 "이번 구간이 μ를 포함할 가능성이 95%"라고 표현하지 않는다. 구간을 만든 뒤 `μ 공개`를 누르면 포함했는지 여부만 확인한다.

### 2. 반복 포함률 확인 모드

목표:
같은 방법으로 표본을 반복해 구간을 만들 때, 그 구간들 중 실제 `μ`를 포함한 비율을 보여준다.

이 모드에서 보여줄 것:

- 여러 신뢰구간
- 실제 `μ` 기준선
- 포함한 구간 수
- 포함하지 못한 구간 수
- 포함 비율
- 선택한 신뢰도

중요:
신뢰도와 포함 비율의 관계는 이 반복 모드에서 설명한다.

```text
신뢰도 95%는 한 구간의 확률이 아니라, 같은 절차로 만든 많은 구간의 장기적 포함 비율을 뜻합니다.
```

## 개념상 가장 중요한 원칙

이 웹앱의 첫 번째 메시지는 다음이어야 한다.

```text
우리는 모집단 전체를 알 수 없습니다.
지금 볼 수 있는 것은 모집단에서 뽑은 표본뿐입니다.
이 표본의 평균 xbar와 표준편차 s로 보이지 않는 모평균 μ의 위치를 추정합니다.
```

추정 과정에서 학생에게 보여주는 값:

- 표본 크기 `n`
- 표본값 목록 또는 점 그림
- 표본평균 `xbar`
- 표본표준편차 `s`
- 표준오차 추정값 `s / sqrt(n)`
- 선택한 신뢰수준
- 오차한계
- 신뢰구간

추정 과정에서 숨겨야 하는 값:

- 실제 모평균 `μ`
- 실제 모표준편차 `σ`
- 모집단 전체 데이터

앱 내부에는 실제 `μ`가 있어도 된다. 단, 실제 `μ`는 구간을 만든 뒤 `μ 공개` 또는 `포함 여부 확인` 버튼을 눌렀을 때만 보여준다.

화면 문구:

```text
실제 μ는 앱이 사후 확인할 때만 사용합니다. 추정할 때는 표본 정보만 사용합니다.
```

## 오개념 방지 문구

다음 표현은 피한다.

```text
이 구간 안에 μ가 95% 확률로 들어 있습니다.
```

대신 다음처럼 표현한다.

```text
95% 신뢰수준이란 같은 방법으로 표본을 반복해 구간을 만들면, 그 구간들 중 약 95%가 실제 μ를 포함한다는 뜻입니다.
```

단일 구간에 대해서는 다음처럼 말한다.

```text
이번 표본으로 만든 하나의 구간입니다. 실제 μ를 포함했는지는 μ를 공개한 뒤 확인할 수 있습니다.
```

신뢰수준 비교 문구:

```text
신뢰수준을 높이면 더 넓은 구간을 만들기 때문에 장기적으로 μ를 포함하는 비율이 높아집니다.
```

단일 구간 화면의 권장 문구:

```text
신뢰수준을 높이면 이번 표본으로 만든 구간도 넓어집니다. 넓어진 구간이 실제 μ를 포함했는지는 μ를 공개한 뒤 확인합니다.
```

## 사용자 조작

### 모집단 상황 선택

모집단 전체 값은 보이지 않게 하되, 어떤 상황에서 표본을 뽑는지는 선택할 수 있게 한다.

- 키 자료
  - 대략 정규형 분포
  - 평균은 숨김
- 시험 점수
  - 40~100 사이의 점수형 자료
  - 약간 왼쪽 또는 오른쪽으로 치우칠 수 있음
- 대기 시간
  - 오른쪽 꼬리가 긴 자료
  - 표본 크기가 작을 때 구간이 흔들리는 것을 보기 좋음
- 사용자 지정 간단 분포
  - 선택 기능
  - 구현 시간이 부족하면 생략 가능

주의:
모집단 선택 이름은 보여주되, 실제 `μ`, `σ`, 전체 히스토그램은 추정 전에는 보여주지 않는다.

### 표본 크기 `n`

- 슬라이더와 빠른 선택 버튼을 함께 둔다.
- 범위: `5`부터 `100`
- 기본값: `20`
- 빠른 선택:
  - `n = 5`
  - `n = 10`
  - `n = 20`
  - `n = 30`
  - `n = 50`
  - `n = 100`

설명 문구:

```text
n이 커지면 보통 s/√n이 작아져 신뢰구간이 좁아집니다.
```

### 신뢰도 또는 신뢰수준

세그먼트 버튼 또는 라디오 버튼으로 제공한다.

- `90%`
- `95%`
- `99%`

기본값: `95%`

설명 문구:

```text
신뢰수준을 높이면 더 조심스럽게 추정하므로 구간이 넓어집니다.
```

화면에서는 학생에게 익숙한 용어를 고려해 `신뢰도(신뢰수준)`처럼 함께 표시해도 된다. 문서와 코드에서는 `confidenceLevel`이라는 변수명을 사용한다.

신뢰도 변경 규칙:

- 현재 표본이 있으면 같은 표본으로 신뢰구간을 다시 계산한다.
- 반복 기록이 있으면 가능하면 같은 표본 기록들을 유지한 채 각 구간만 다시 계산한다.
- 이렇게 해야 같은 표본 조건에서 신뢰도가 높아질 때 구간 폭과 포함 비율이 어떻게 바뀌는지 공정하게 비교할 수 있다.

### 실행 버튼

- `표본 1개 추출`
- `신뢰구간 만들기`
- `μ 공개`
- `새 표본으로 다시 추정`
- `100번 반복`
- `초기화`

버튼 흐름:

1. `표본 1개 추출`
   - 숨겨진 모집단에서 표본 `n`개를 뽑고 표본값을 보여준다.
2. `신뢰구간 만들기`
   - `xbar`, `s`, `s/√n`, 오차한계, 신뢰구간을 차례로 보여준다.
3. `μ 공개`
   - 실제 `μ` 기준선을 표시하고, 이번 구간이 `μ`를 포함했는지 확인한다.
4. `새 표본으로 다시 추정`
   - 같은 모집단에서 새 표본을 뽑아 새 신뢰구간을 만든다.
5. `100번 반복`
   - 같은 방식으로 표본 100개를 반복 추출하고 신뢰구간 100개를 누적 표시한다.

선택 기능:

- `반복 횟수` 선택을 둘 수 있다.
  - `50번`
  - `100번`
  - `500번`
- 기본값은 `100번`으로 둔다.
- 수업 시연에서는 100번이 빠르고, 포함 비율의 장기적 경향을 더 안정적으로 보려면 500번이 좋다.

  - `μ 공개됨`
  - `반복 실험 결과 표시 중`

### 표본 관찰 영역

표본값 `n`개를 점 그림 또는 작은 카드로 보여준다.

필수 표시:

- 표본값들
- 표본평균 `xbar` 기준선
- 표본값들이 `xbar` 주변에 흩어져 있음을 보여주는 간단한 표시

선택 표시:

- 표본표준편차 `s`를 설명하는 짧은 시각화
  - 예: `xbar ± s` 구간을 표본 점 그림에 옅은 띠로 표시

주의:
이 영역에는 모집단 평균 `μ` 기준선을 처음부터 표시하지 않는다.

### 추정 계산판

계산은 단계적으로 드러난다.

```text
1. 표본평균 xbar = ...
2. 표본표준편차 s = ...
3. 표준오차 추정값 = s / √n = ...
4. 오차한계 = 임계값 × s / √n = ...
5. 신뢰구간 = xbar ± 오차한계 = [L, U]
```

학생이 공식의 역할을 볼 수 있도록, 계산판에는 `xbar`, `s`, `n`, 신뢰수준이 각각 구간에 어떤 영향을 주는지 짧게 표시한다.

예:

```text
s가 클수록 표본이 많이 흩어져 있어 구간이 넓어집니다.
n이 클수록 s/√n이 작아져 구간이 좁아집니다.
신뢰수준이 높을수록 임계값이 커져 구간이 넓어집니다.
```

### 신뢰구간 숫자축

가로 숫자축에 다음을 표시한다.

- 표본평균 `xbar`: 굵은 점 또는 세로선
- 신뢰구간 `[L, U]`: 굵은 막대
- 구간 양끝 `L`, `U`: 숫자 라벨
- 실제 `μ`: 처음에는 숨김
- `μ 공개` 후 실제 `μ`: 빨간 기준선 또는 별도 표식

`μ 공개` 후:

- `μ`가 구간 안에 있으면:
  - 구간 막대 색: 파랑 또는 초록
  - 문구: `이번 구간은 실제 μ를 포함했습니다.`
- `μ`가 구간 밖에 있으면:
  - 구간 막대 색: 빨강
  - 문구: `이번 구간은 실제 μ를 포함하지 못했습니다.`

### 반복 신뢰구간 기록판

`100번 반복`을 누르면 여러 신뢰구간을 세로로 쌓아 보여준다.

패널 제목:

```text
반복 포함률 확인: 같은 방법으로 만든 여러 신뢰구간
```

표시 규칙:

- 각 줄: 표본 하나로 만든 신뢰구간 하나
- 가운데 점 또는 세로선: 해당 표본의 `xbar`
- 구간 막대: `[L, U]`
- 실제 `μ` 기준선: 반복 기록판에서는 표시한다.
- `μ`를 포함한 구간: 파랑 또는 초록
- `μ`를 포함하지 못한 구간: 빨강

누적 결과 표시:

- 만든 구간 수
- `μ` 포함 구간 수
- 포함 비율
- 선택한 신뢰수준

예:

```text
100개 구간 중 94개가 μ를 포함했습니다. 포함 비율 94%
```

해석 문구:

```text
매번 만든 구간 하나하나가 μ를 반드시 포함하는 것은 아닙니다. 하지만 같은 방법을 많이 반복하면 포함 비율은 선택한 신뢰수준 근처로 다가갑니다.
```

반복 기록 저장 방식:

- 각 반복에서 최소한 다음 값을 저장한다.
  - `sample`
  - `xbar`
  - `s`
  - `n`
- 신뢰도 변경 시 저장된 `xbar`, `s`, `n`으로 구간을 다시 계산한다.
- 그러면 같은 표본 묶음에 대해 90%, 95%, 99% 신뢰구간을 공정하게 비교할 수 있다.

주의:
반복 횟수가 100번이면 포함 비율이 선택한 신뢰도와 정확히 일치하지 않을 수 있다. 이 차이는 오류가 아니라 무작위 표본 추출의 자연스러운 흔들림이다.

## 수학 모델 또는 계산식

모집단:

```text
population = [x1, x2, ..., xN]
```

실제 모평균:

```text
mu = sum(population) / N
```

주의:
`mu`는 앱 내부 검증용 값이다. 추정 계산에는 사용하지 않는다.

표본:

```text
sample = [x1, x2, ..., xn]
```

표본평균:

```text
xbar = sum(sample) / n
```

표본표준편차:

```text
s = sqrt(sum((xi - xbar)^2) / (n - 1))
```

표준오차 추정값:

```text
estimatedSE = s / sqrt(n)
```

신뢰구간:

```text
margin = criticalValue * estimatedSE
lower = xbar - margin
upper = xbar + margin
```

포함 여부:

```text
containsMu = lower <= mu && mu <= upper
```

## 적용 조건과 한계

화면 또는 해석 문구에 다음 전제를 짧게 포함한다.

```text
이 추정은 모집단에서 표본이 무작위로 뽑혔다고 가정합니다.
```

수업용 한계 설명:

- 표본이 편향되어 있으면 신뢰구간도 믿기 어렵다.
- 표본 크기 `n`이 작고 모집단이 심하게 치우쳐 있으면 포함 비율이 신뢰도와 차이 날 수 있다.
- 반복 실험의 포함 비율은 무작위로 흔들리므로, 100번 반복 결과가 정확히 90%, 95%, 99%가 되지 않아도 정상이다.
- 이 웹앱은 개념 이해용 시뮬레이션이며, 실제 통계 분석에서는 자료 수집 방법과 표본의 대표성을 함께 검토해야 한다.

학생용 짧은 문구:

```text
좋은 추정은 좋은 표본에서 시작합니다. 표본이 한쪽으로 치우치면 구간 계산이 정확해도 추정은 흔들릴 수 있습니다.
```

## 임계값 처리

모표준편차 `σ`를 모르는 상황이므로 표본표준편차 `s`로 표준오차를 추정한다.

권장 구현:

- 고등학교 교과서 공식에 따라 정규 임계값(z)을 사용한다.
- 신뢰수준과 임계값:
  - 90% → z = 1.645
  - 95% → z = 1.960
  - 99% → z = 2.576

오차한계 계산식:

```text
margin = z × s / sqrt(n)
```

화면에는 다음 문구를 넣는다.

```text
모표준편차 σ를 모르기 때문에 표본표준편차 s로 표준오차를 추정합니다.
(교과서 공식: x̄ ± z × s/√n)
```

화면 설명:

```text
모표준편차 σ를 모르기 때문에 s를 사용합니다.
표본이 클수록 s가 σ에 가까워져 추정이 안정됩니다.
```

구현 참고 (화면에 표시하지 않아도 됨):
통계학에서는 σ를 모를 때 t 임계값(자유도 df = n − 1)이 이론적으로 더 정확하다.
고등학교 교과서는 z 임계값을 표준 공식으로 가르치므로, 이 앱도 z값을 기본으로 사용해 교과서와 일치하게 한다.

## 검증 방법

기본 동작:

- 페이지를 열었을 때 모집단 평균이 보이지 않는지 확인한다.
- `표본 1개 추출` 전에는 신뢰구간 계산 버튼이 비활성인지 확인한다.
- 표본 추출 후 `xbar`, `s`, `n`이 표시되는지 확인한다.
- `신뢰구간 만들기`를 누른 뒤 `[L, U]`가 숫자축에 표시되는지 확인한다.
- `μ 공개`를 누르기 전에는 `μ` 기준선이 보이지 않는지 확인한다.
- `μ 공개` 후 포함/미포함 판정이 표시되는지 확인한다.

수치 검증:

- 표본값들의 평균이 표시된 `xbar`와 일치하는지 확인한다.
- 표본표준편차가 분모 `n - 1`로 계산되는지 확인한다.
- `estimatedSE = s / sqrt(n)`이 맞는지 확인한다.
- `margin = criticalValue * estimatedSE`가 맞는지 확인한다.
- `lower = xbar - margin`, `upper = xbar + margin`이 맞는지 확인한다.
- 같은 표본에서 90%, 95%, 99% 순서로 신뢰구간 폭이 커지는지 확인한다.
- 95% 신뢰수준일 때 임계값 z = 1.960을 사용하는지 확인한다. (교과서 기준 z값)

상태 검증:

- 신뢰수준만 바꾸면 현재 표본을 유지한 채 구간이 다시 계산되는지 확인한다.
- 반복 기록이 있는 상태에서 신뢰수준만 바꾸면 같은 반복 표본 기록으로 구간과 포함 비율이 다시 계산되는지 확인한다.
- 표본 크기나 모집단 상황을 바꾸면 기존 표본과 반복 기록이 초기화되는지 확인한다.
- `100번 반복` 중 버튼 중복 클릭으로 기록이 겹치지 않는지 확인한다.
- `초기화`가 표본, 구간, `μ` 공개 상태, 반복 기록을 모두 초기화하는지 확인한다.

개념 검증:

- 화면 어디에도 "이번 구간 안에 μ가 95% 확률로 있다"는 식의 문구가 없어야 한다.
- "같은 방법으로 반복해 만든 구간들의 장기적 포함 비율"이라는 설명이 있어야 한다.
- 실제 `μ`는 추정 계산에 쓰지 않고 사후 확인에만 쓰인다는 문구가 있어야 한다.
- 단일 표본 화면에서는 포함 여부를 `μ 공개` 후에만 말하고, 신뢰수준의 의미는 반복 포함률 화면에서 설명해야 한다.
- 무작위 표본이라는 전제와 편향된 표본의 한계가 안내되어야 한다.

레이아웃 검증:

- 데스크톱에서 표본, 계산판, 신뢰구간 숫자축, 반복 기록판이 한 흐름으로 읽히는지 확인한다.
- 모바일에서 각 영역이 세로로 쌓이고 버튼 텍스트가 넘치지 않는지 확인한다.
- 반복 기록판이 길어져도 페이지 전체가 지나치게 밀리지 않는지 확인한다.
