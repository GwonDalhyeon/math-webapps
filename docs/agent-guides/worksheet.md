# 학습지 Markdown 작성 지침

이 문서는 `worksheet.md`, `worksheet-short.md` 등 학습지 Markdown 파일을 만들거나 수정할 때 읽는다.

## 위치와 이름

학습지는 해당 웹앱 폴더 안에 둔다. 학습지 Markdown을 제작할 때는 학생용 학습지와 예시답안이 있는 교사용 학습지를 항상 함께 만든다.

```text
probability-statistics/sample-proportion-event-readiness/worksheet.md
probability-statistics/sample-proportion-event-readiness/worksheet-teacher.md
probability-statistics/sample-proportion-event-readiness/worksheet-short.md
probability-statistics/sample-proportion-event-readiness/worksheet-short-teacher.md
```

웹앱과 직접 연결되는 학습지는 `tasks/`가 아니라 웹앱 폴더에 둔다.

파일명 규칙:

- 기본 학생용: `worksheet.md`
- 기본 교사용: `worksheet-teacher.md`
- 짧은 학생용: `worksheet-short.md`
- 짧은 교사용: `worksheet-short-teacher.md`
- 다른 변형을 만들 때도 학생용과 교사용 파일명을 같은 접두어로 맞춘다.
- 기존 학생용 학습지를 수정할 때는 대응하는 교사용 학습지도 함께 수정한다.
- 대응하는 교사용 학습지가 없으면 새로 만든다.

예:

```text
worksheet-practice.md
worksheet-practice-teacher.md
```

## 수식 작성 규칙

학습지 Markdown에 들어가는 모든 수식은 반드시 LaTeX 형식으로 작성한다.

인라인 수식:

```md
모비율은 $p=0.7$이고 표본비율은 $X/n$이다.
```

별도 줄 수식:

```md
$$
X \sim B(n,p)
$$
```

텍스트식 대신 LaTeX를 사용한다.

```md
<!-- 권장 -->
$E(X)=np=20 \times 0.7=14$

<!-- 피함 -->
E(X) = np = 20 x 0.7 = 14
```

표 안에서도 수식은 LaTeX로 쓴다.

```md
| 준비 인원 $c$ | 준비 부족 조건 | 확률 |
|---:|---|---:|
| 14명 | $X \ge 15$ | $P(X>14)=0.416$ |
```

## 학습지 성격

학생용 학습지는 학생이 웹앱을 보면서 생각을 기록하는 도구이다. 교사용 학습지는 같은 문항 구조를 유지하되 예시답안, 계산 과정, 수업 중 짚을 점을 포함한다.

좋은 학습지는 다음 흐름을 가진다.

```text
상황 이해 -> 간단한 계산 -> 웹앱 관찰 -> 비교 -> 개념 정리
```

학생은 주로 보는 목적이고, 교사가 설명하면서 진행하는 수업이라면 다음을 우선한다.

- 계산은 쉬운 값으로 제한한다.
- 관찰 기록 칸을 충분히 둔다.
- 웹앱에서 바뀌는 요소와 학습지 질문을 직접 연결한다.
- 정답 계산보다 개념 구분을 묻는다.
- 마지막에는 한 문장 정리를 쓰게 한다.

## 학생용과 교사용 구성

학생용 학습지:

- 답을 쓰는 빈칸을 둔다.
- 계산 과정 일부를 남겨 학생이 완성하게 한다.
- 관찰 기록과 자기 생각을 쓰는 공간을 둔다.
- 수업 중 바로 읽기 쉬운 문장으로 쓴다.

교사용 학습지:

- 학생용 문항 순서와 번호를 유지한다.
- 각 빈칸에 들어갈 예시답안을 제공한다.
- 서술형 문항에는 가능한 학생 답안 예시를 1개 이상 제시한다.
- 교사가 강조할 개념이나 오개념 주의점을 짧게 덧붙일 수 있다.
- 예시답안도 수식은 모두 LaTeX 형식으로 작성한다.

교사용 표기 예:

```md
**예시답안:** $E(X)=20 \times 0.7=14$명이므로 평균적으로 14명이 참석한다.

**지도 포인트:** 평균 14명은 매번 14명이 참석한다는 뜻이 아니라 장기적 평균이라는 점을 강조한다.
```

## 기본 구성

권장 구성:

1. 제목, 이름, 학번
2. 상황 이해
3. 기본값으로 계산하기
4. 웹앱 1회 관찰
5. 여러 번 실행한 결과 기록
6. 조건을 바꾸어 비교
7. 개념 정리
8. 선택 탐구 또는 다음 차시 연결

예시:

```md
# 표본비율과 행사 준비 학습지

이름: ____________  학번: ____________

## 1. 상황 이해

신청자 한 명이 실제 참석할 확률은 $p=0.7$이고, 신청자 수는 $n=20$명이다.

1. 평균 참석자 수를 계산하라.

$$
E(X)=np=20 \times 0.7=\underline{\hspace{2cm}}
$$
```

## 빈칸과 서술형 답안

계산 빈칸:

```md
$$
20 \times 0.7=\underline{\hspace{2cm}}\text{명}
$$
```

짧은 서술:

```text
____________________________________________________________
```

표 기록:

```md
| 반복 횟수 | 문제 발생 횟수 | 문제 발생 비율 |
|---:|---:|---:|
| 10회 | ____회 | ____% |
| 100회 | ____회 | ____% |
```

## 문항 작성 기준

- 한 문항에서 묻는 개념은 하나만 둔다.
- 계산값은 수업 중 암산하거나 짧게 계산할 수 있게 한다.
- `왜 그런가?` 문항에는 힌트를 제공할 수 있다.
- 용어는 웹앱 화면의 용어와 일치시킨다.
- 모비율, 표본비율, 표본 크기, 반복 횟수를 혼동하지 않게 한다.

## 표본과 분포 학습지 주의점

표본 관련 학습지는 다음 구분을 명확히 한다.

- `n`: 한 표본 또는 한 행사 안의 대상 수
- 반복 횟수: 표본이나 행사를 몇 번 만들었는지
- 모비율 또는 모평균: 장기적으로 기대되는 모집단의 특성
- 표본비율 또는 표본평균: 한 번의 표본에서 계산된 값
- 표본분포: 표본 통계량을 반복해서 모은 분포

## PDF 변환을 고려한 작성

- 한 페이지 버전은 문항 수를 줄이고 표를 간결하게 한다.
- 두 페이지 버전은 섹션 구분을 명확히 하되 여백이 과하지 않게 한다.
- Markdown 원본은 학습용 구조를 유지하고, PDF 레이아웃 압축은 변환 과정에서 처리한다.
- PDF를 만들 때도 학생용 PDF와 교사용 PDF를 같은 형식 기준으로 함께 만든다. 페이지 수는 사용자가 지정한 목표를 우선하고, 교사용 답안 때문에 같은 페이지 수가 어려우면 최종 응답에 알린다.
- PDF 변환이 예정된 경우 `pdf-export.md`도 읽는다.
