치환적분: 같은 함수값, 다른 누적 속도

0 ≤ x ≤ 1에서 f(2x)를 적분하면 함수 f의 값은 0 ≤ u ≤ 2를 모두 지나간다. 하지만 u의 두 칸이 x의 한 칸에 압축되므로, 같은 높이를 더해도 넓이는 절반이 된다.

u = ax, dx = du / a
위 그래프 누적
0.000
∫₀ᵘ f(t) dt
아래 그래프 누적
0.000
∫₀ˣ f(as) ds
관계
아래 = 위 / 2
같은 높이, 가로폭만 1/a
입력 이동 속도 a 2.00
현재 x 위치 0.65
직사각형 개수 12

관찰 포인트

위쪽은 u축에서 f(u)를 누적한다. 현재 u = ax이다.

아래쪽은 x축에서 f(ax)를 누적한다. 같은 함수값을 보지만 가로폭이 줄어든다.

같은 높이의 직사각형을 비교하면 Δx = Δu / a이므로 넓이도 1/a배가 된다.

웹앱과 함께 푸는 핵심 문제

먼저 관찰하고, 식은 나중에 정리하도록 구성한 문항이다.

1. 같은 값을 지나가는데 왜 넓이가 다를까?
  1. a = 2로 두고 위쪽 누적값과 아래쪽 누적값을 비교하시오.
  2. 아래쪽 넓이가 위쪽 넓이의 몇 배인지 쓰시오.
  3. 그 이유를 "함수값의 높이"와 "가로폭"이라는 말을 사용하여 설명하시오.
2. 직사각형 한 칸으로 설명하기
  1. 위쪽 그래프의 한 직사각형 폭을 Δu라고 하자.
  2. 같은 함수값에 대응하는 아래쪽 직사각형의 폭 Δx를 구하시오.
  3. 두 직사각형의 넓이 비를 설명하시오.
3. 계산으로 확인하기
  1. f(t) = t + 1일 때 ∫₀¹ f(2x) dx를 구하시오.
  2. ∫₀² f(t) dt를 구하시오.
  3. 두 값의 관계가 웹앱에서 본 결과와 일치하는지 확인하시오.
4. 일반화하기
  1. u = ax이면 du와 dx의 관계를 쓰시오.
  2. ∫₀¹ f(ax) dx를 u에 대한 적분으로 바꾸시오.
  3. 치환적분에서 왜 1/a가 붙는지 말로 설명하시오.
