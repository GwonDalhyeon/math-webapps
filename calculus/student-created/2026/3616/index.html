선택한 부분 또는 주제 미분방정식을 이용하여 생물의 개체수 변화를 확인하는 로지스틱 함수
관련 미적분 개념 자연로그, 지수함수, 미분방정식
웹앱 제작 의도 미분방정식을 이용하여 각 조건을 조절함에 따라 변화하는 생물의 개체수 변화를 실시간으로 관찰할 수 있다.
사용 방법 안내 미입력

2. 개념 설명 영역

주제 선택 이유: S자 모양의 로지스틱 함수를 구하기 위해 각 단계의 한계점을 파악하고 저항 계수와 같은 조건들을 추가하여 원하는 함수 모양에 도달해나가는 과정이 인상깊었으며 이 주제를 통하여 미분방정식의 유용함을 느끼고 이러한 미분방정식을 통해 미래를 예측 가능하게 하며 이러한 예측 가능성이 물고기의 적당한 어획량, 생물의 환경에 따른 개체수 변화 등을 알 수 있게하는 큰 가치에 대해 알게되어 이 부분을 선택하였다.

핵심 원리: 생물의 개체수가 계속해서 증가하는 것이 아닌 다양한 환경적 요인들에 인하여 도달하는 한계점이 존재한다.

4. 조작 영역

t = 0일때의 처음 개체수를 조절

먹이, 한정된 공간에 따른 환경 저항 계수를 조절

  • 개체수가 계속해서 증가하지 않고 먹이 또는 공간과 같은 환경적 요인으로 인하여 포화되는 상태가 존재하며 이러한 예측을 통하여 물고기의 어획량, 생물의 별종 방지 대책과 같은 여러 방안들을 알아낼 수 있다.
