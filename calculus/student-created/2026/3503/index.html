정적분

웹앱 제작 의도
해당 웹앱은 푸리에 급수를 활용하여 복합 파동 내에 포함된 사인 성분과 코사인 성분을 찾아내는 과정을 보여준다.

사용 방법 안내
사인파의 주파수(f₁), 코사인파의 주파수(f₂), 푸리에 계수의 차수(n)를 조절해 보세요. 그래프와 푸리에 계수 aₙ, bₙ이 어떻게 변하는지 확인할 수 있습니다.

개념 설명

복합 파동이 여러 개의 사인파와 코사인파의 합으로 이루어져 있다고 가정하자. 푸리에 급수에서는 정적분을 이용하여 특정 차수 n의 성분이 실제로 포함되어 있는지 확인할 수 있다.

n번째 성분이 포함되어 있으면 해당 계수(aₙ 또는 bₙ)가 크게 나타나고, 포함되어 있지 않으면 적분값이 0이 되어 사라진다.

aₙ = (1/π)∫f(x)cos(nx)dx
bₙ = (1/π)∫f(x)sin(nx)dx

이 웹앱은 정적분을 이용해 복합 파동 속에 어떤 주파수 성분이 들어 있는지 찾아내는 원리를 시각적으로 보여준다.

시각화 영역

aₙ : 0
bₙ : 0

읽으면서 흥미로웠던 점: aₙ과 bₙ에 n값을 넣어가다 보면 복합 파동에 존재하지 않는 성분은 0이 되어 사라지고, 실제로 존재하는 성분만 남게 되어 복합 파동을 구성하는 단순 파동을 찾아낼 수 있다는 점이 흥미로웠다.
