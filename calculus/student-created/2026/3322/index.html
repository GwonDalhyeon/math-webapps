웹앱 제작 의도
이 웹앱은 인공지능이 미분을 이용하여 손실함수의 최솟값을 찾고 학습하는 과정을 시각적으로 보여준다.
사용 방법
① 현재 위치 슬라이더를 움직인다.
② 학습 1단계 버튼을 누른다.
③ 자동 학습 버튼을 눌러 AI가 최솟값을 찾는 과정을 관찰한다.

개념 설명

인공지능은 예측값과 실제값의 차이인 손실함수(Loss Function)를 최소로 만드는 방향으로 학습한다.

미분을 이용하면 현재 위치에서 함수의 기울기(도함수)를 구할 수 있다. 인공지능은 이 기울기를 이용해 오차가 감소하는 방향으로 이동한다.

기울기가 0에 가까워질수록 손실함수의 최솟값에 가까워지며, 인공지능은 점점 더 정확한 답을 찾게 된다.
