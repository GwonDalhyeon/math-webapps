웹앱 제작 의도: 이계도함수로 극값을 찾고 그래프의 개형 예측하기
사용 방법 안내: [x값 입력창]에 원하는 분석 위치(0부터 500까지)를 입력하면 해당 범위까지의 예시 주가 그래프가 즉시 업데이트됩니다. 생성된 그래프 영역을 마우스로 붙잡고 좌우로 드래그하면 전체 그래프(x축 범위 500)를 웹 화면 이동 없이 편리하게 넘겨볼 수 있습니다. 또한, 그래프 선 위의 특정 지점에 마우스 포인터를 올리면(Hover) 그 시점의 함수 값과 이계도함수 값을 연동하여 실시간 분석 결과를 나타냅니다.

개념 설명 영역

읽으면서 생긴 질문/생각: 미적분을 통해 그래프의 극값을 판정하고 개형을 예측하는 것을 주식에 연계하여 주식의 하락과 상승을 예측할 수 있다는 것이 흥미로웠고, 실제로 적용이 가능한가라는 의문이 생겼다.

이계도함수를 이용한 극값 판정 원리: 연속이고 이계도함수가 존재하는 함수에서 도함수 f'(x) = 0인 지점을 찾은 후, 그 지점의 이계도함수 f''(x)의 부호를 확인하여 극값을 판정합니다.

  • f'(x) = 0 이고 f''(x) < 0 이면 극대 (위로 볼록한 최고점 부근으로 하락 변동 예측 가능)
  • f'(x) = 0 이고 f''(x) > 0 이면 극소 (아래로 볼록한 최저점 부근으로 상승 변동 예측 가능)

시각화 영역 (0 ≤ x ≤ 500, 1 ≤ y ≤ 50)

조작 영역

* 그래프 영역을 마우스로 드래그하여 x축 방향 이동이 가능합니다.
