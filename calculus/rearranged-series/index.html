교대조화급수의 재배열로 임의의 상수 c에 가까워지기

양의 항과 음의 항을 따로 나열한 뒤 클릭하여 더해 보며, 부분합이 목표값 c에 가까워지도록 재배열하는 웹앱입니다. 클릭은 직접 할 수도 있고, 자동으로 진행하게 할 수도 있습니다.

수학 개념 설명

원래 급수는 다음과 같습니다.

an = (-1)n+1 · 1/n,    Σan = 1 - 1/2 + 1/3 - 1/4 + 1/5 - ...

양의 항만 모으면 1, 1/3, 1/5, 1/7, ..., 음의 항만 모으면 -1/2, -1/4, -1/6, -1/8, ... 입니다.

이 앱에서는 목표값 c를 정한 뒤, 부분합이 c보다 작으면 다음 양의 항을 더하고, 부분합이 c보다 크면 다음 음의 항을 더하는 방식으로 진행합니다. 그러면 부분합이 목표값을 번갈아 넘나들면서 점점 가까워지는 모습을 볼 수 있습니다.

S < c 이면 다음 양의 항 추가
S > c 이면 다음 음의 항 추가

이 앱은 “덧셈의 순서를 바꾸면 부분합의 움직임이 달라진다”는 점을 시각적으로 보여 주기 위한 탐구용 모델입니다.

교대조화급수 재배열 부분합 수렴

조작 영역

직접 클릭해도 되고, “한 단계 자동 선택” 또는 “자동 실행 시작”으로 목표값에 가까워지는 과정을 볼 수도 있습니다.

현재 상태

목표값 c 1.200000
현재 부분합 S 0.000000
오차 |S - c| 1.200000
사용한 항의 개수 0
다음 양의 항 1
다음 음의 항 -1/2
자동 추천 현재 부분합이 c보다 작으므로 다음 양의 항을 더하는 것이 좋습니다.
